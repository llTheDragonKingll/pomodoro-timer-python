# ⏳ Pomodoro Timer (Python Tkinter)

A simple Pomodoro timer built using Python and Tkinter to help manage work and break sessions.

---

## 🚀 Features

* ⏱️ Work and break intervals
* 🔁 Automatic session switching
* ✔️ Tracks completed sessions
* 🖥️ Clean and minimal GUI

---

## 🛠️ Tech Stack

* Python
* Tkinter

---

## 📂 Project Structure

```
pomodoro-timer-python/
│
├── main.py
└── tomato.png
```

---

## ⚙️ How to Run

```
python main.py
```

---

## 📌 Pomodoro Technique

* 25 minutes work
* 5 minutes short break
* 15–20 minutes long break

---

## 💡 Notes

* Includes a tomato icon image for UI
* Timer automatically handles session flow
* GUI updates dynamically during countdown
